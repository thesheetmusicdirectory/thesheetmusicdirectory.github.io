About The Guitar Directory

Welcome to The Guitar Directory – your all-in-one resource for legendary guitar tabs, audio, and videos.

Whether you're a beginner or a seasoned player, you'll find a curated collection of classic and modern songs, each with easy access to tabs and audio.

This project was created and is maintained by Mattia Lobrano as a passion for music and web development. I strive to keep the site clean, fast, and easy to use for all guitarists.

If you have feedback, suggestions, or want to contribute, please get in touch!
